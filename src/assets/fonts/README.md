# README for Fonts

This directory contains the font files used in the English A1 eBook project. The fonts are chosen to enhance readability and provide an engaging learning experience for users.

## Font Usage

- Ensure that the fonts are properly linked in the CSS files to maintain consistent styling across the eBook.
- Fonts should be loaded efficiently to optimize performance and user experience.

## Future Updates

As the project evolves, additional fonts may be added to support new themes or design updates. Please document any new fonts added to this directory in this README file.
